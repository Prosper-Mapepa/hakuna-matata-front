# RSVP Frontend

Next.js frontend for Hakuna Matata Party RSVP System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The frontend will run on `http://localhost:3000` by default.

## Pages

- `/` - Event landing page with RSVP form
- `/success` - Success page after RSVP submission
- `/admin` - Admin dashboard (password protected)

## Environment Variables

Make sure the backend is running on `http://localhost:3001` or update the API URLs in the pages.
